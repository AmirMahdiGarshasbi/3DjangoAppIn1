# 3DjangoAppIn1
This repository contains three different app in one project due to the lack of internet and not installing Django over and over again.

settings.py file is in diary directory.

Consider that each app is one of its own and these three apps have nothing in common. Just follow the urls after runnig server with
>python manage.py runserver

entries was the first app and you can write your diary in it.

todo is an app that is in Persian and you can list the works you should do daily.

and blog is a simple blog app with the use of django generiv views.

Have fun!
